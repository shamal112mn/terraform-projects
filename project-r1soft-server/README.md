# terraform-hw3  

### r1SOFT backup package management

 ```
 Create a folder called "r1soft".
 In that r1soft folder create a file that provisions r1soft server 
 using terraforms provisioner feature. Terraform has to create a r1soft repo on the remote system and install r1soft server 
 that works on port 80 and 443.

More details how to install follow link below:

http://wiki.r1soft.com/display/ServerBackup/Install+Server+Backup+Manager+on+Linux

```
